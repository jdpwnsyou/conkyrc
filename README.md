conkyrc
=======
This is a configuration file for Conky.


It includes real time monitoring for:

Kernel Version

Uptime

Battery

Updates needed

CPU temp

CPU usage by core

RAM stats

SWAP stats

TOP

Date and Time

Up/Downspeed graphs and stats

ESSID 

Local/Public IP

IP Region 

Hard drive stats(/, /home, /dev)

HDD temp
